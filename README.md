# vuejsremotedev

Vue Js Template For Remote Development (on VPS / VPC)

allow port :8080 on your instance/machine

## Ubuntu:
```
sudo ufw allow 8080
```

## Run Your Project
```
docker-compose up
```

## Access UI on your browser
```
0.0.0.0:8080

[YOUR MACHINE IP]:8080

[YOUR INSTANCE DOMAIN]:8080
```
